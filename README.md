# eBook Library

eBook Library is a simple web application for managing and browsing a collection of e-books.  
It was built as a learning project using C#, HTML, CSS, and JavaScript.

## Tech Stack

- **Backend / App:** C# (.NET / Visual Studio solution `LabCommunictionProj.sln`)
- **Frontend:** HTML, CSS, JavaScript

## Features (Overview)

- Web interface for an online eBook library
- Pages for browsing books and viewing details
- Personal page showing books the user has bought or borrowed
- Shopping cart for managing selected books before checkout
- Basic layout and styling for practicing full-stack web development
